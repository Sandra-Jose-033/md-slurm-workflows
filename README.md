# md-slurm-workflows
SLURM-based molecular dynamics workflows for GPU-accelerated GROMACS simulations

**Workflow**
- NVT equilibration
- NPT equilibration
- Production MD

**Tested on**
- NVIDIA V100
- SLURM scheduler
- Apptainer containerized GROMACS

**Notes**
- Update the container path before submission
- Adjust SLURM directives for your cluster
